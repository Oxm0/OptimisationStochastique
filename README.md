# Application d'Optimisation Stochastique

## Description
Cette application permet d'optimiser les paramètres de divers processus industriels en tenant compte des incertitudes. Elle inclut des fonctionnalités pour importer des données, classer les processus, optimiser les paramètres, visualiser les résultats et créer des moyens de production.
